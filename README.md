# Hvutils

A set of small utilities.

* get_mac : returns the MAC address of the interface.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hvutils'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hvutils

## Usage

    Hvutils.get_mac('eth0') => mac address as string
